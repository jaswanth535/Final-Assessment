Responsive design is the term that describes flexibility, which means if we write proper code for once and it makes the page look greater everywhere. For example, the code we have written on the computer makes look good in it and it also has to look good on iPad, iPhones, and android mobiles as well. The Responsive design brings a lot of changes to the website based on the size of a screen and also the device by which we are using to view the website. So, this responsive design mainly responds to the screen size that we are viewing. So instead of creating a lot of different websites for different devices our responsive design rearranges the elements to fit in any screen and by doing so weâ€™ll get more visitors and that also leads to more happy customers. The one language that makes responsive design easier is CSS. We can always add responsive later but it will be a lot more effective than if we add it at the beginning itself. So finally, we can tell that responsive design is writing code once to make our website look great.

One way to create a proper responsive design is by using media queries. So, these media queries allow us to create CSS rules based on different parameters like size, viewport, device type, etc. So, we can arrange our layout to suit a different variety of screen sizes or devices. We can create a media query that targets just mobile screen sizes and we can also create a whole set of CSS rules that are going to target screen sizes. We can include these media queries in the CSS that we are using.
